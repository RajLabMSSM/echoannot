# Order loci by UCS size, or alphabetically

Order loci by UCS size, or alphabetically

## Usage

``` r
order_loci(
  dat,
  merged_DT,
  by_UCS_size = FALSE,
  descending = TRUE,
  verbose = FALSE
)
```

## Examples

``` r
if (FALSE) { # \dontrun{
merged_DT <- echodata::get_Nalls2019_merged()
merged_DT <- order_loci(
    dat = merged_DT,
    merged_DT = merged_DT,
    descending = FALSE
)
} # }
```
