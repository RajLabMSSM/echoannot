# GoShifter p-value histograms

Plot histograms of GoShifter p-values for each annotation source. Two
p-value columns are shown side-by-side: the manually computed `pval` and
the GoShifter-reported `GS_pval`.

## Usage

``` r
GOSHIFTER_histograms_pvals(GS_results)
```

## Arguments

- GS_results:

  A `data.table` of GoShifter results containing columns `Annotation`,
  `pval`, and `GS_pval`.

## Value

A `ggplot` object (also printed).
