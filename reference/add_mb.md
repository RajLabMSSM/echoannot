# Add Mb

Add column containing genomic coordinate in units of megabasepairs
("Mb").

## Usage

``` r
add_mb(dat, pos_col = "POS")
```

## Arguments

- dat:

  Data.

- pos_col:

  Genomic position column name.

## Examples

``` r
if (FALSE) { # \dontrun{
dat <- echodata::BST1
dat2 <- add_mb(dat = dat)
} # }
```
