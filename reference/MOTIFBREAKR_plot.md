# Plot motifbreakR results

Plot motif disruption results generated by
[MOTIFBREAKR](https://rajlabmssm.github.io/echoannot/reference/MOTIFBREAKR.md).

## Usage

``` r
MOTIFBREAKR_plot(
  mb_res,
  mb_filter = NULL,
  rsid = NULL,
  effect = c("strong", "weak"),
  results_dir = file.path(tempdir(), "results"),
  height = 3,
  width = 7,
  verbose = TRUE
)
```

## Source

[Publication](https://pubmed.ncbi.nlm.nih.gov/26272984/)
[GitHub](https://github.com/Simon-Coetzee/MotifBreakR) [Viewport
bug](https://github.com/Simon-Coetzee/motifBreakR/issues/31)

## Arguments

- mb_res:

  Results generated by
  [MOTIFBREAKR](https://rajlabmssm.github.io/echoannot/reference/MOTIFBREAKR.md),
  in [GRanges](https://rdrr.io/pkg/GenomicRanges/man/GRanges-class.html)
  format.

- mb_filter:

  [GRanges](https://rdrr.io/pkg/GenomicRanges/man/GRanges-class.html)
  object used to filter `mb_res` before plotting.

- rsid:

  Character vector of RSIDs to plot. If `NULL`, all RSIDs in `mb_res`
  are used.

- effect:

  Character vector of effect strengths to include (e.g. `"strong"`,
  `"weak"`).

- results_dir:

  Directory where results should be saved as a file named:
  *\<results_dir\>/\_genome_wide/motifbreakR/motifbreakR_results.rds*.
  If `NULL`, results will not be saved to disk.

- height:

  the height of the device.

- width:

  the width of the device.

- verbose:

  Print messages.

## Value

Named list of motif plot paths.

## Details

*Notes:*\

- Saving as a PDF seems to work much better than PNG format (at least
  when using grDevices).
