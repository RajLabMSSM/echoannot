# Get cell type-specific superenhancer data

Brain cell-specific epigenomic data from Nott et al. (2019).

## Usage

``` r
NOTT2019_superenhancers(dat)
```

## Source

Nott et al. (2019)
([doi:10.1126/science.aay0793](https://doi.org/10.1126/science.aay0793)
)
