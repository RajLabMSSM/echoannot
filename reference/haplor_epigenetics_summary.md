# Summarise `HaploR` annotations

Summarise `HaploR` annotations

## Usage

``` r
haplor_epigenetics_summary(
  merged_results,
  tissue_list = c("BRN", "BLD"),
  epigenetic_variables = c("Promoter_histone_marks", "Enhancer_histone_marks")
)
```
