# Download SNP-wise annotations from RegulomeDB

Download SNP-wise annotations from RegulomeDB

## Usage

``` r
haplor_regulomedb(snp_list, timeout = 500, chunk_size = NA, verbose = TRUE)
```

## Source

[HaploR](https://cran.r-project.org/web/packages/haploR/vignettes/haplor-vignette.html)

## Arguments

- verbose:

  Print messages.
