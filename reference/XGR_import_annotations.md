# Download XGR annotations

Download XGR annotations

## Usage

``` r
XGR_import_annotations(
  gr.snp,
  anno_data_path = file.path("annotations", paste0("XGR_", lib.name, ".rds")),
  lib.name,
  save_xgr = TRUE,
  annot_overlap_threshold = 5
)
```
